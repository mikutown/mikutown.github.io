编写一个静态方法lg()

1.1.14接受一个整型参数N，返回不大于log2(N)的最大整数。不要使用Math库

  • 答案
package top.leafii.practice.charpter01;
import edu.princeton.cs.algs4.StdIn;
import edu.princeton.cs.algs4.StdOut;
public class Practice02 {
    public static int lg(int N){
        int s = 0;
        while (N/2>=1){
            N/=2;
            s++;
        }
        return s;
    }
    public static void main(String[] args) {
        int a = 0;
      while (true){
          a = StdIn.readInt();
          System.out.println(lg(a));
      }
    }
}

编写一个静态方法 histogram()

1.1.15 接受一个整型数组 a[] 和一个整数 M 为参数并返回一个大小为 M 的数组，其中第 i 个元素的值为整数 i 在参数数组中出现的次数。如果 a[] 中的值均在 0 到 M-1之间，返回数组中所有元素之和应该和 a.length 相等。

英文题目原文：

1.1.15 Write a static method histogram() that takes an array a[] of int values and an integer M as arguments and returns an array of length M whose i th entry is the number of times the integer i appeared in the argument array. If the values in a[] are all between 0 and M–1, the sum of the values in the returned array should be equal to a.length.

  • 分析

    干就完了！

  • 答案

package top.leafii.practice.charpter01;

import com.sun.xml.internal.bind.v2.runtime.output.FastInfosetStreamWriterOutput;
import edu.princeton.cs.algs4.StdIn;
import edu.princeton.cs.algs4.StdOut;
import edu.princeton.cs.algs4.StdRandom;

public class Practice02 {
    public static int[] histogram(int[] a,int M){
        int[] b = new int[M];
        for (int i = 0; i < b.length; i++) {
            b[i] = 0;
        }
        for (int i = 0; i < a.length; i++) {
            if(a[i]<=M)
                b[a[i]]++;
        }
        int sum = 0;
        for (int i = 0; i < b.length; i++) {
            System.out.print(b[i] + " ");
            sum += b[i];
        }
        System.out.println();
        if(sum ==a.length)
            System.out.println("输入的数组中的值均在0到M-1之间");
        else
            System.out.println("输入的数组中的值不都在0到M-1之间");

        return b;
    }
    public static void main(String[] args) {
        int size = 0;
        int M = 0;
      while (true){
          System.out.println("请输入数组大小：");
          size = StdIn.readInt();
          int[] a = new int[size];
          System.out.println("请输入数组的值：");
          for (int i = 0; i < a.length; i++) {
              a[i] = StdIn.readInt();
          }
          System.out.println("请输入M的值:");
          M = StdIn.readInt();
          if(M>a.length)
              histogram(a,M);
      }
    }
}

public static int[] histogram(int[] a,int M)
{
    int[] b = new int[M];
    int n = 0;
    int m = 0;
    for(int i = 0; i < M; i++)
    {
        for(int j = 0; j < a.length ; j++ )
        {
            if(i == a[j])
            {
                n++;
            }
            b[i] = n;
        }
        n = 0;
    }

    for(int i = 0 ; i < M ; i++)
    {
        m = m + b[i];
    }

    return b;
}

请看以下递归函数:

public static int mystery(int a, int b) {
   if (b == 0)     return 0;
   if (b % 2 == 0) return mystery(a+a, b/2);
   return mystery(a+a, b/2) + a;
}

1.1.18mystery(2, 25) 和 mystery(3, 11) 的返回值是多少?给定正整数 a 和 b，mystery(a,b)计算的结果是什么?将代码中的 + 替换为 * 并将 return 0 改为 return 1，然后回答相同 的问题。

  • 分析

    • 当代码中为+时
    1. mystery(2,25) :

    a=2,b=25,因为b!=0且b%2!=0，所以返回mystery(4,12) + 2

    ​ mystery(4,12) 中 a=4,b=12,因为b!=0 ，b%2==0，所以return mystery(8,6)

    ​ mystery(8,6)中，a=8,b=6 return mystery(16,3)

    ​ mystery(16,3)中，a=16,b=3, return mystery(32,1)+16

    ​ mystery(32,1)中，a=32,b=1,return mystery(64,0)+32

    ​ mystery(64,0)==0

    所以mystery(2,25)=2+16+32+0=50

  • 答案：

public static int mystery(int a, int b) {
    if (b == 0)
        return 0;
    if (b % 2 == 0)
        return mystery(a+a, b/2);
    return mystery(a+a, b/2) + a;
}
public static int mystery1(int a, int b) {
    if (b == 0)
        return 1;
    if (b % 2 == 0)
        return mystery1(a*a, b/2);
    return mystery1(a*a, b/2) * a;
}

public static void main(String args[]) {  
    System.out.println(mystery(2,25));  // 输出50
    System.out.println(mystery(3,11));  //输出33
    System.out.println(mystery1(2,25));  // 输出33554432
    System.out.println(mystery1(3,11));  //输出177147
}

package top.leafii.practice.charpter01;

import com.sun.xml.internal.bind.v2.runtime.output.FastInfosetStreamWriterOutput;
import edu.princeton.cs.algs4.StdIn;
import edu.princeton.cs.algs4.StdOut;
import edu.princeton.cs.algs4.StdRandom;

public class Practice02 {
    public static long F(int N) {
//        if(N==0)
//            return 0;
//        if(N==1)
//            return 1;
        if(N>1){
            int[] a = new int[N+1];
            a[0] = 0;
            a[1] = 1;
            for (int i = 2; i < a.length; i++) {
                a[i] = a[i-2] + a[i-1];
            }
            return a[N];
        }
        return 0;
    }

        for (int N = 0; N < 100; N++)
            StdOut.println(N + " " + F(N));
    }
}

编程题

1.1.21 编写一段程序，从标准输入按行读取数据，其中每行都包含一个名字和两个整数。然后用printf()打印一张表格，每行的若干列数据包括名字、两个整数和第一个整数除以第二个整数的结果，粗确到小数点后三位。可以用这种程序将棒球球手的击球命中率或者学生的考试分数制成表格。

  • 答案
package top.leafii.practice.charpter01;

import edu.princeton.cs.algs4.StdIn;
import edu.princeton.cs.algs4.StdOut;

public class Practice02 {
    public static double F(int N) {
        if(N==0){
            return 1;
        }
        else{
            return Math.log(N)+F(N-1);
        }
    }

    public static void main(String[] args) {
        int row = 0;//定义行号
        StdOut.print("请输入数据总的行数:");
        row = StdIn.readInt();//从键盘获取数据行数
        String Name = "Name";
        String V1 = "Value1";
        String V2 = "Value2";
        String V3 = "Value3";
        String [] str = new String[row];
        String [] num1 = new String[row];
        String [] num2 = new String[row];
        double [] num3 = new double[row];
        for (int i = 0; i < str.length; i++) {
            str[i] = "";
            num1[i] = "";
            num2[i] = "";
            num3[i] = 0;
        }
        for (int i = 0; i < row; i++) {
            StdOut.println("请输入名字 数字1 数字2 格式:\"Jack 100 100\"");
            str[i] = StdIn.readString();
            num1[i] = StdIn.readString();
            num2[i] = StdIn.readString();
            num3[i] = Double.parseDouble(num1[i])/Double.parseDouble(num2[i]);
        }
        //遍历
        StdOut.println("******************************");
        System.out.printf("%-8s%-8s%-8s%-8s",Name,V1,V2,V3);
        System.out.println();
        //StdOut.println("Name    V1         V2        V3");
        for (int i = 0; i < str.length; i++) {
            System.out.printf("%-8s%-8s%-8s%-8.3f",str[i],num1[i],num2[i],num3[i]);
            System.out.println();
        }
        System.out.println("******************************");

    }
}
