编写一个程序

1.1.3

从命令行得到三个整数参数。如果它们都相等则打印 equal，否则打印 not equal。

package top.leafii.practice.charpter01;

import edu.princeton.cs.algs4.In;
import edu.princeton.cs.algs4.StdIn;
import edu.princeton.cs.algs4.StdOut;

/***
 * @author leafii
 *
 */
public class practice01 {
    public static void main(String[] args) {
        /**
         * 编写一个程序，从命令行得到三个整数参数。
         * 如果它们都相等则打印 equal，
         * 否则打印 not equal。
         */
        while (true){ //为了能多重输入，便于测试哦
            int[] a = new int[3];
            for (int i = 0; i < a.length; i++) {
                a[i] = StdIn.readInt();
            }
            if(a[0] == a[1] && a[1] == a[2] && a[0] == a[2])
            {
                StdOut.println("equal");
            }
            else
            {
                StdOut.println("not equal");
            }
        }
    }
}

运行结果

12 23 34
not equal
11 11 11
equal

下列语句各有什么问题(如果有的话)?

1.1.4

a. if (a > b) then c = 0;
b. if a > b { c = 0; }
c. if (a > b) c = 0;
d. if (a > b) c = 0 else b = 0;

a. if (a > b) then c = 0;   //java中不存在then关键字
b. if a > b { c = 0; }   //没有括号，应该是(a > b)
c. if (a > b) c = 0;    //没问题，不过最好加括号
d. if (a > b) c = 0 else b = 0; //有问题 c=0没有分号，还有二义性，没加花括号

编写一段程序

1.1.5

如果 double 类型的变量 x 和 y 都严格位于 0 和 1 之间则打印 true，否则打印 false

package top.leafii.practice.charpter01;

import edu.princeton.cs.algs4.In;
import edu.princeton.cs.algs4.StdIn;
import edu.princeton.cs.algs4.StdOut;

/***
 * @author leafii
 *
 */
public class practice01 {
    public static void main(String[] args) {
        /**
         如果 double 类型的变量 x 和 y 都严格位于 0 和 1 之间则打印 true，
         否则打印 false
         */
        while (true){
            double x = StdIn.readDouble();
            double y = StdIn.readDouble();
            StdOut.println(x>=0 && x <=1 && y>=0 && y<=1);
        }
    }
}

运行结果：

0.1
0.89
true
1.2
0.1
false
0.3
3.4
false
