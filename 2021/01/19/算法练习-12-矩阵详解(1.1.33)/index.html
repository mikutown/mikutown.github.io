题目

1.1.33矩阵库。编写一个Matrix库并实现以下API

编写一个测试用例，从标准输入读取矩阵并测试所有方法。

  • 分析

在书中第一次出现矩阵的例子是：

int N = a.length;
double[][] c = new double[N][N];
for (int i = 0; i < N; i++)
   for (int j = 0; j < N; j++)
   { // Compute dot product of row i and column j.
      for (int k = 0; k < N; k++)
         c[i][j] += a[i][k]*b[k][j];
}

这段代码被用来计算矩阵的乘积。

由此可见 用java创建矩阵，需要你创建一个二维数组。

使用Java实现矩阵
由 m × n 个数aij排成的m行n列的数表称为m行n列的矩阵，简称m × n矩阵。而行数与列数都等于n的矩阵称为n阶矩阵或n阶方阵。

向量(Vector)
本题中除了矩阵还提到了向量，那向量又是什么，跟矩阵有何区别与联系呢？
其实我们可以把向量当作一个点(a,b)。它与矩阵的区别是：
向量可以用矩阵表示,且有时特殊矩阵就是向量。简言之就是矩阵包含向量。

点乘(dot)
在数学中，数量积(dot product; scalar product，也称为点积、点乘)是接受在实数R上的两个向量并返回一个实数值标量的二元运算。它是欧几里得空间的标准内积。
坐标表示：已知两个非零向量 a = (x1, y1), b = (x2, y2)，则有a 点乘 b = x1x2 + y1y2，即两个向量的数量积等于它们对应坐标的乘积的和。

乘积
两个矩阵的乘法仅当第一个矩阵A的列数和另一个矩阵B的行数相等时才能定义。乘积矩阵的行数等于左边矩阵的行数,乘积矩阵的列数等于右边矩阵的列数 。

package top.leafii.practice.charpter01;

public class Matrix {
    /**
     * 向量点乘
     * @param x    double类型的一维数组
     * @param y    double类型的一维数组
     * @return     两个向量点乘的乘积
     */
    public static double dot(double[] x, double[] y){
        double a = 0.0;
        if(x.length!=y.length){
            return a;
        }else{
            for (int i = 0; i < x.length; i++) {
                a += x[i] * y[i];
            }
        }
        return a;
    }

    /**
     * 矩阵和矩阵之积
     * @param a     矩阵a
     * @param b     矩阵b
     * @return      矩阵a，b的积矩阵c
     */
    public static double[][] mult(double[][] a, double[][] b){
        if(a[0].length != b.length){
            return new double[1][1];
            //抛出异常
        }
        double[][] c = new double[a[0].length][b[0].length];
        for (int i = 0; i < a.length; i++) {
            for (int j = 0; j < a[0].length; j++) {
                for (int k = 0; k < a[0].length; k++) {
                    c[i][j] += a[i][k] * b[k][j];
                }
            }
        }
        return c;
    }

    /**
     * 转制矩阵
     * @param a    要转制的矩阵a
     * @return     返回结果
     */
    public static double[][] transpose(double[][] a) {
        double temp = 0;
        for (int i = 0; i < a.length; i++) {
            for (int j = 0; j < a[0].length; j++) {
                temp = a[i][j];
                a[i][j] = a[j][i];
                a[j][i] = temp;
            }
        }
        return a;
    }

    /**
     * 　矩阵和向量之积
     * @param a    矩阵a
     * @param x    向量x
     * @return     矩阵a和向量x的乘积
     */
    public static double[] mult(double[][] a, double[] x){
        double[] c = new double[a.length];
        if(a[0].length != x.length){
            //抛出异常
            return new double[1];
        }
        for (int i = 0; i < a.length; i++) {
            for (int j = 0; j < a[0].length; j++) {
                c[i] += a[i][j] * x[j];
            }
        }
        return c;
    }

    /**
     * 向量和矩阵之积
     * @param x      向量x
     * @param a      矩阵a
     * @return       向量x和矩阵a的积
     */
    public static double[] mult(double[] x, double[][] a){
        double[] c = new double[x.length];
        if(x.length != a[0].length){
            //抛出异常
            return new double[1];
        }
        for (int i = 0; i < x.length; i++) {
            for (int j = 0; j < a[0].length; j++) {
                c[i] += a[i][j] * x[j];
            }
        }
        return c;
    }
}
