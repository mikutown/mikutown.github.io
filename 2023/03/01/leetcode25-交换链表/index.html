leetcode25.交换链表

2023-03-01- 2023-03-06
618- 2m
leetcode算法

题目

给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。

示例 1：

输入：head = [1,2,3,4]
输出：[2,1,4,3]

示例 2：

输入：head = []
输出：[]

示例 3：

输入：head = [1]
输出：[1]

提示：

  • 链表中节点的数目在范围 [0, 100] 内
  • 0 <= Node.val <= 100

解答

  1. 直接进行交换，思路如下：

当链表为空或者只有一个节点时，无需任何操作，直接返回head，否则就定义指针cur和before，初始值分别为head和NULL。当链表中至少有两个节点时进行循环，在循环里定义一个指针tmp，用于指向cur的下一个节点，接下来进行两个节点的交换操作，此时判断before指针是否为空，若不为空则将before的next指向交换完毕的tmp指针，然后更新cur指针的值，继续循环直到链表中未得到交换的元素数小于2停止，最后返回头节点head。C++实现如下：

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* swapPairs(ListNode* head) {
        // 如果链表为空或只有一个节点，直接返回头指针
        if (head == NULL || head->next == NULL) return head;

        // 定义指针 cur 和 before，初始值分别为 head 和 NULL
        ListNode* cur = head;
        ListNode* before = NULL;

        // 当链表中至少有两个节点时进行循环
        while(cur != NULL && cur->next != NULL) {
            // 定义指针 tmp，将其指向 cur 的下一个节点
            ListNode* tmp = cur->next;

            // 将 cur 的下一个节点指向 tmp 的下一个节点
            cur->next = tmp->next;

            // 将 tmp 的下一个节点指向 cur，完成交换
            tmp->next = cur;

            // 如果不是第一次交换，则将 before 的下一个节点指向 tmp
            if (before != NULL) before->next = tmp;

            // 如果 cur 是链表头节点，则将 head 指向 tmp
            if (cur == head) head = tmp;

            // 将 before 指向 cur，cur 指向下一个节点，进行下一次循环
            before = cur;
            cur = cur->next;
        }

        // 返回交换后的链表头指针
        return head;
    }
};
